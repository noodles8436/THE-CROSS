addXY_inList(x, y)[source]

It is a function that stores coordinates input through screen clicks.

Parameters

  • x (int) – The x-coordinate entered through the screen click.

  • y (int) – The y-coordinate entered through the screen click.

Note

the x, y must be equal to or greater than zero.

Note

the x, y must be equal to or less than the maximum size of the screen.

Warning

If the number of already stored coordinates is more than 4, the already stored coordinates are removed except for the currently received coordinates.

This function is called when a screen click event occurs. This function is called when a screen click event occurs, and receives the coordinates clicked on the screen and stores the coordinates in the list (self.touch_list).

Parameters

  • event (opencv_event) – Types of events that occurred.

  • x (int) – The x-coordinate entered through the screen click.

  • y (int) – The y-coordinate entered through the screen click.

  • flags – It is a parameter to comply with the form defined in OpenCV.

  • param – It is a parameter to comply with the form defined in OpenCV.

Warning

The function should not be called except for calls by click events.
