Server module

Server.startServer()[source]

This function starts the Detection Server to receive an image from the Client and send the detection result of the Object Detection to the Client.

To run Server.py, enter IP and PORT information of server in the console window to Open Object Detection Server

Note

The default IP of the Server is localhost.

Note

The default PORT of the Server is 7777.

Note

DO NOT CLOSE PROGRAM UNTIL Socket Opened Message printed

Example:

Run Server.py on the console in the following format: >> python Server.py –ip=XX.XX.XX.XX –port=XXX
